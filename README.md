# ConceptDynamics.github.io

Welcome to my personal GitHub Pages website!

## About This Site

This is a simple, clean personal website hosted on GitHub Pages. It showcases my work and provides a central hub for my online presence.

## Features

- 📱 Responsive design that works on all devices
- 🎨 Modern, professional styling
- ⚡ Fast loading with static HTML
- 🔧 Easy to customize and extend

## Getting Started

To customize this site for yourself:

1. **Edit `index.html`** - Update the welcome message, bio, and links
2. **Update `_config.yml`** - Change the title, description, and author
3. **Add more pages** - Create new `.html` or `.md` files

## Local Development

To preview changes locally:

```bash
# Install Jekyll (requires Ruby)
gem install jekyll bundler

# Navigate to your repo directory
cd ConceptDynamics.github.io

# Run the local server
jekyll serve

# Visit http://localhost:4000
```
